# 声乐练习音高曲线

这是一个可直接部署到 GitHub Pages 的静态前端页面，用于实时采集麦克风输入、检测音高并绘制音高曲线。

## 使用方式

1. 软件启动后会先显示模式选择页，包含“音高曲线 / 频谱图 / 音域训练模式（暂无功能）/ 音准评分（暂无功能）”四个选项。
2. 点击“音高曲线”进入原本实时检测界面；点击“频谱图”会直接进入频谱图视图。
3. 进入后可点击右上角“返回”按钮回到初始模式选择页。
4. 点击“开始检测”，浏览器会请求麦克风权限，允许后即可看到实时音高曲线、频率与音名。
5. 点击“停止”结束本次采集。
6. 侧边栏“音高算法”可切换不同的音高检测算法（AMDF / 自相关 / 频域 HPS / YIN）。
7. 侧边栏“音高刻度”可选择移动刻度、固定刻度（50–500 Hz）或对数刻度（动态）。
8. 侧边栏“共振峰”开关开启后，会同时显示 F1/F2 数字与对应的曲线。
9. 侧边栏“显示模式”可切换音高曲线与实时频谱图视图。
10. 侧边栏“窗口缩放”可以调节音高窗口占屏幕的大小比例。
11. 侧边栏“音准目标线”可开启/关闭目标线，并设置目标音高（例如 300 Hz）与实时曲线叠加对比。
12. 在桌面端可将鼠标移到主窗口边缘或四角并拖拽，像 Windows 一样手动调整窗口大小。
13. 在“显示模式”中可选择频谱叠加（音高曲线 / 共振峰曲线），用于在频谱图上同步观察。
14. 侧边栏“显示音量条”可以控制左侧音量刻度条的显示。
15. 录音：点击“开始录音”/“停止录音”后，可在“离线分析”中选择分析最近录音或下载到本地。
16. 伴奏播放：在侧边栏上传伴奏文件后可播放/暂停/停止并调整音量。建议使用耳机减少伴奏被麦克风拾取。
17. 音准模式评估：先上传伴奏（作为参考音高），再完成一段录音，点击“音准模式评估”即可计算平均音高偏差（cents）；偏差较小会显示“音高准确”，偏差较大会显示“跑调”。

> 音高灵敏度优化：实时检测已改为“自适应能量阈值”，并降低了起音门槛（更低起音置信度 + 更短起音帧数），在轻声或起音阶段更容易显示音高曲线。

## 保存/导出

- **保存本次（CSV）**：会导出最近 `maxHistorySeconds` 秒内的有效音高点。
  - CSV 列包含 `timestampMs`（相对开始检测的毫秒）、`frequencyHz`（保留 1 位小数）、`note`。
- **导出截图（PNG）**：将当前 canvas 画面保存为 PNG 文件。

以上两个按钮在未开始检测或没有有效音高数据时会置灰。

## 上传音频离线分析

侧边栏提供“上传音频”功能，支持常见音频格式（如 wav/mp3）。流程如下：

1. 点击“上传音频”选择文件，页面会显示“数据源：音频文件”并开始离线分析。
2. 分析过程中会显示进度百分比；完成后可导出 CSV/PNG。
3. 点击“清除文件/返回实时”即可回到实时麦克风模式。

默认最大分析时长为 300 秒（约 5 分钟），超过时会提示是否继续分析全片。若解码失败，请尝试使用 wav/mp3 格式。
